🔒 Permissions & Security

✅ No Manual Permissions Required!

This extension uses activeTab permissions, which means:

  • ✅ Works on any website without manual permission grants
  • ✅ Only accesses pages when you click the extension icon
  • ✅ More secure - no broad host permissions needed
  • ✅ Privacy-focused - no background access to your browsing

How It Works

1. Navigate to website

Go to any website you want to scrape (e.g., products.example.com)

2. Click extension icon

The extension automatically gets temporary access to that specific tab

3. Start scraping

Use Smart Picker or run your saved jobs - it just works!

Help & Documentation

🚀 Quick Start

  1. Open Sidebar: Click the Vibe Scraper icon in Chrome toolbar
  2. Create Job: Enter a Job ID (e.g., "products")
  3. Smart Picker: Click 🎯 Smart Picker button, then click any item on the page
  4. Select Mode: Choose from 5 scraping modes (Single Page, Pagination, Infinite Scroll, Load More)
  5. Start Scraping: Click ▶️ Start Scraping button
  6. Download: CSV downloads automatically when complete!

🎯 Smart Features

  • Smart Picker: Automatically detects fields by clicking any item on the page
  • Smart Detect: Click 🎯 next to Next/Load More buttons to auto-detect selectors
  • Real-time Progress: Visual overlay shows current item, page, and total count
  • Stop Anytime: Click ⏹️ Stop button to gracefully terminate scraping

📄 Scraping Modes

Single Page
Scrapes only the current page. Perfect for one-time extractions.
Button Pagination
Auto-clicks Next buttons. Works on SPAs and dynamic sites. Use 🎯 Detect for automatic selector.
Query Parameter Pagination
Fetches pages via URL parameters (?page=2). Fast background fetching. Works on static HTML sites.
Infinite Scroll
Auto-scrolls to load more content. Perfect for social feeds and dynamic lists.
Load More Button
Auto-clicks Load More buttons. Use 🎯 Detect for automatic selector.

🎨 Beautiful Toggle Switch

When using Pagination mode, toggle between:

  • Button (LEFT): Click Next buttons - works on all sites
  • Query Param (RIGHT): Fast background fetch - for ?page=X URLs

Choose Button for SPAs, Query Param for static sites!

⚙️ Advanced Settings

Use this Settings page for:

  • Managing saved jobs

For quick scraping, use the sidebar instead!

💡 Best Practices

  • Start with Max Pages = 2 to test your setup
  • Use Smart Picker instead of writing selectors manually
  • Check console (F12) for detailed debug logs
  • Always review a website's robots.txt and Terms of Service
  • Use appropriate delays to avoid overloading servers
  • Respect website rate limits and access restrictions
  • Only scrape publicly available information

🐛 Troubleshooting

No items found?
Use Smart Picker to auto-detect the correct selector.
Pagination not working?
Use 🎯 Smart Detect next to the Next Button field. If it's a SPA, use Button mode instead of Query Param.
Query param returns no data?
Site might require JavaScript. Try Button Pagination mode instead.
Need help?
Check console logs (F12) for detailed error messages.

📝 Example Configuration

{
  "jobId": "products-example",
  "startUrl": "https://example.com/products",
  "listing": {
    "itemSelector": ".product-card",
    "fields": [
      { 
        "name": "title", 
        "selector": ".product-title", 
        "attr": "text"
      },
      { 
        "name": "price", 
        "selector": ".product-price", 
        "attr": "text"
      },
      { 
        "name": "url", 
        "selector": "a.product-link", 
        "attr": "href"
      }
    ]
  },
  "pagination": {
    "enabled": true,
    "type": "button",
    "nextButtonSelector": "a.next",
    "maxPages": 10
  }
}
